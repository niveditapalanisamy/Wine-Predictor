# Wine-Quality-Predictor
This project is based on predicting the quality of wine based on the parameters
  * fixed acidity 
  * volatile acidity
  * citric acid
  * residual sugar
  * chloride
  * free sulphur oxide
  * total suphur di oxide
  * density
  * pH
  * sulphates
  * alcohol
## Libraries used
  * numpy
  * pandas
  * seaborn
  * matplotlib
  * StandardScaler
  * Decision Tree Classifier
  * Ada Boost Classifier
